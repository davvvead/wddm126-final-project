About

WDDM 126 Project Part 2: Branching, Conflicts, Debugging & Pull Request Workflow. This project demonstrates Git feature branching, handling merge conflicts, debugging code with DevTools, and submitting pull requests for code review. This is part 2 of the final project, focusing on branching, conflicts, debugging, and pull request workflow.

Lab 10 Learning Outcomes

  • Master feature branching with Git
  • Resolve merge conflicts effectively
  • Debug JavaScript using browser DevTools
  • Create and review Pull Requests
